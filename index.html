Fresco使用

一、使用前准备工作

1.在module的build.gradle中加入    compile 'com.facebook.fresco:fresco:0.8.1'
2.权限声明：<uses-permission android:name="android.permission.INTERNET"/>

二、使用

1.初始化 在Application类onCreate中添加初始化代码 Fresco.initialize(this);，然后在AndroidManifest.xml中声明 android:name=".MyApplication"
2.1 加载一张普通图片
    2.1.1 加入控件
    <com.facebook.drawee.view.SimpleDraweeView
        android:id="@+id/my_image_view"
        android:layout_width="20dp"
        android:layout_height="20dp"
        fresco:placeholderImage="@drawable/my_drawable"
      />
    2.1.2 加载图片
        Uri uri = Uri.parse("https://raw.githubusercontent.com/facebook/fresco/gh-pages/static/fresco-logo.png");
        SimpleDraweeView draweeView = (SimpleDraweeView) findViewById(R.id.my_image_view);
        draweeView.setImageURI(uri)
    2.1.3 其他
    Fresco会替你完成:
        1.显示占位图直到加载完成；
        2.下载图片；
        3.缓存图片；
        4.图片不再显示时，从内存中移除；
    2.1.4 属性值
        fresco:actualImageScaleType="focusCrop"//图片的ScaleType方式
        fresco:placeholderImage="@color/wait_color"
        fresco:placeholderImageScaleType="fitCenter"//占位图和其ScaleType方式
        fresco:failureImage="@drawable/error"
        fresco:failureImageScaleType="centerInside"//加载失败的图片和ScaleType方式
        fresco:retryImage="@drawable/retrying"
        fresco:retryImageScaleType="centerCrop"//重新加载的图片和ScaleType方式
        fresco:progressBarImage="@drawable/progress_bar"
        fresco:progressBarImageScaleType="centerInside"
        fresco:progressBarAutoRotateInterval="1000"//progressbar
        fresco:roundAsCircle="false"//圆形图
        fresco:roundedCornerRadius="1dp"//圆角大小
        fresco:roundTopLeft="true"
        fresco:roundTopRight="false"
        fresco:roundBottomLeft="false"
        fresco:roundBottomRight="true"//四个角分别设置圆角方式
        fresco:roundWithOverlayColor="@color/corner_color"
        fresco:roundingBorderWidth="2dp"//圆形图片边框大小
        fresco:roundingBorderColor="@color/border_color"//圆形图片边框颜色
